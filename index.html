Vanilla JS Prompt modals

Simple alert function with endless possibilities.

Default close button prompt



show_prompt('Default close button', 'Do you want to color heading?', false, undefined, 'Yes', `document.getElementById('demo-heading').style.background = '#04AA6D';document.getElementById('modal-alert').style.display = 'none'`)

No close button prompt



show_prompt('No close button prompt', 'Do you want to reload?', true, '', 'Refresh', `location.reload();`)

Custom buttons prompt



show_prompt('Default close button', 'Visit larrindell.com?', false, 'No', 'Yes', `window.open('https://larrindell.com');document.getElementById('modal-alert').style.display = 'none'`)

Simple alert prompt



show_prompt('Simple alert prompt', 'Only close button')
